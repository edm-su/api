# API edm.su

## Требования

* Python >= 3.10
* PostgreSQL >= 9.6
* Poetry
* Meilisearch >= 0.26

## Установка

```shell
poetry install
alembic upgrade head
```

## Запуск

```shell
./entrypoint.sh
```
